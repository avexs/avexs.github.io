Personal Website

HTML/CSS/Bootstrap/fontAwesome

The purpose of this project was to learn more about web development and the importance of creating a personal brand. I purposely avoided using a template when I began this project, since I wanted to have the full experience of designing and programming a responsive website. I knew I would not learn as much if I had chosen a template site; and although the aesthetics of the site maybe be lacking as a result of this decision, the payoff was completely worth it.

If you are interested in viewing the source and bugs for this site, please click on the computer icon above to visit the github for this project.

RGB Rubix Cube

Three-dimensional rubix cube using RBG leds and an Arduino

This project began when I attended my very first hackathon, for which I unfortunately showed up on the last day and was unable to finish in time. The idea for this project was to create arrays of leds which would act as each face of the cube. Using switches, the user would be able to rotate portions of the cube. The plan for this project was to first implement the cube using an arduino as the logic controller, and to then later syntehsize the project using gate-level logic.
